# Anonymous Supplementary Materials

This repository hosts the anonymous supplementary website for the paper:

**IssueExec: A Test-Driven Approach for Localizing Software Engineering Issues**

## How to access
Open the GitHub Pages site for this repository (link provided in the submission).

## Notes on anonymity
- Do not include author names, affiliations, emails, or identifying URLs.
- Avoid uploading git history from a personal repository.
- Ensure PDF metadata does not contain author information.
